    Part B -- Cloudinary CDN test:
    - Use a plain `<img>` tag with a Cloudinary demo URL to verify the CDN path works:
      ```html
      <img
        src="https://res.cloudinary.com/demo/image/upload/w_400,f_auto,q_auto/sample.jpg"
        alt="Cloudinary CDN test image"
        width="400"
        height="267"
        loading="lazy"
        class="rounded-lg shadow-md w-full max-w-lg"
      />
      ```
    - The `f_auto` and `q_auto` Cloudinary transforms serve WebP/AVIF automatically based on browser support
    - This uses the Cloudinary public `demo` cloud for verification. The actual retreat cloud name will be used in Phase 3.

      </div>
    </div>
