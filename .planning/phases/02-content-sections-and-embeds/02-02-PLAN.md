    **Lazy-load script** (placed AFTER the HTML markup -- Pitfall 2):
    ```html
    <script>
      const iframe = document.getElementById('calendar-iframe') as HTMLIFrameElement | null;
      if (iframe) {
        const observer = new IntersectionObserver(
          (entries) => {
            if (entries[0].isIntersecting) {
              const src = iframe.dataset.src;
              if (src) iframe.src = src;
              observer.disconnect();
            }
          },
          { rootMargin: '200px 0px' }
        );
        observer.observe(iframe);
      }
    </script>
    ```
    rootMargin 200px starts loading before the user scrolls to it, avoiding loading flash.
