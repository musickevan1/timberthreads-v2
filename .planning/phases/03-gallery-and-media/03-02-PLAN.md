**1. Install PhotoSwipe:**

This adds `photoswipe` (v5.4.x) to package.json dependencies.

**2. Import PhotoSwipe CSS in global.css:**
Add to `src/styles/global.css` AFTER the existing `@import "tailwindcss"` line:
```css
@import "photoswipe/dist/photoswipe.css";
```

CRITICAL: Do NOT import PhotoSwipe CSS inside an Astro `<script>` tag — it will not work (see Research Pitfall 1). The CSS must be imported through the stylesheet pipeline.

After installing, verify the actual CSS file path by checking `node_modules/photoswipe/dist/` — the import path might be `photoswipe/dist/photoswipe.css` or similar. Use whatever path exists in the installed package.

**3. Add caption styles to global.css:**
Add after the photoswipe import:
```css
/* PhotoSwipe caption overlay */
.pswp__custom-caption {
  position: absolute;
  bottom: 0;
  left: 0;
  right: 0;
  padding: 16px 24px;
  background: linear-gradient(transparent, rgba(0, 0, 0, 0.5));
  color: #fff;
  font-size: 0.875rem;
  text-align: center;
  pointer-events: none;
}
```

The `pswp__custom-caption` class name is auto-generated by PhotoSwipe from the `name: 'custom-caption'` property (it prepends `pswp__` and converts to the class).

**4. Add PhotoSwipe initialization script to Gallery.astro:**
Add a `<script>` tag (Astro bundled module, NOT `is:inline`) at the bottom of Gallery.astro:

```typescript
import PhotoSwipeLightbox from 'photoswipe/lightbox';

// Initialize one lightbox per category gallery to keep navigation within categories
document.querySelectorAll<HTMLElement>('[data-pswp-gallery]').forEach((gallery) => {
  const lightbox = new PhotoSwipeLightbox({
    gallery,
    children: 'a',
    pswpModule: () => import('photoswipe'),

    // Zoom behavior for mobile (pinch-to-zoom, double-tap zoom)
    initialZoomLevel: 'fit',
    secondaryZoomLevel: 1.5,
    maxZoomLevel: 3,
  });

  // Caption support via custom UI element
  // Reads plain text from data-pswp-caption attribute on slide anchor elements
  lightbox.on('uiRegister', function () {
    lightbox.pswp!.ui!.registerElement({
      name: 'custom-caption',
      order: 9,
      isButton: false,
      appendTo: 'root',
      onInit: (el: HTMLElement) => {
        lightbox.pswp!.on('change', () => {
          const currSlideElement = lightbox.pswp!.currSlide!.data.element as HTMLElement | undefined;
          const caption = currSlideElement?.getAttribute('data-pswp-caption') || '';
          el.textContent = caption;
          el.style.display = caption ? '' : 'none';
        });
      },
    });
  });

  lightbox.init();
});
```

This approach:
- Initializes one lightbox per `[data-pswp-gallery]` container (per-category navigation, not cross-category)
- PhotoSwipe natively handles: navigation arrows, X of Y counter (built-in), Escape to close, click outside to close, close button, arrow key navigation, swipe on mobile, pinch-to-zoom, double-tap zoom
- Captions read from `data-pswp-caption` attribute using `textContent` (safe, no HTML injection)
- `data-pswp-srcset` on the anchor tags (from Plan 01) gives PhotoSwipe responsive full-size images automatically
